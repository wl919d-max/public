
# 入库单条码打印功能 PRD

## 1. 文档信息

| 项目 | 内容 |
| --- | --- |
| 需求名称 | 入库单条码打印 |
| 所属模块 | 库存管理 / 入库管理 |
| 文档版本 | V1.0 |
| 当前状态 | 待评审 |
| 适用端 | Web 管理后台 |
| 关联原型 | `inbound-barcode-print-prototype.html` |
| 核心原则 | 一物一标、标签内容由模板驱动、打印数量可调整、只累计已打印数量、打印过程可追溯 |

## 2. 背景与问题

仓库完成采购入库、其他入库或生产入库后，通常需要为实物粘贴物料标签，用于后续上架、盘点、调拨、出库和追溯。

当前若依赖人工录入标签内容或在第三方软件中重复维护物料资料，会产生以下问题：

1. 物料编码、名称、规格、批次等信息重复录入，效率低且容易出错。
2. 无法准确记录本次入库物料累计已打印数量，操作员难以判断标签打印进度。
3. 不同打印机、标签尺寸和标签模板缺少统一管理。
4. 打印结果未与业务单据关联，发生重复打印或标签异常时难以追溯。
5. 打印能力若仅写死在入库页面，后续销售出库、仓位、资产等场景会重复建设。

因此，需要建设与入库单联动的条码标签打印能力，同时将模板、预览、打印通道、任务和记录等通用能力抽象为可复用的打印中心。

## 3. 产品目标

1. 支持从入库单详情页快速打印当前单据的物料标签。
2. 支持从入库单列表选择多张单据批量打印。
3. 展示本次入库数量、累计已打印数量和参考待打印数量。
4. 支持按物料自由调整本次打印数量，允许超过参考待打印数量。
5. 支持标签模板选择，并按模板读取标签尺寸、条码/二维码规则及打印布局，支持打印前预览。
6. 支持主流标签打印机，并对打印服务和设备状态进行检测。
7. 记录每次打印任务、打印明细、操作人和执行结果。
8. 沉淀可复用的标签模板、打印预览、打印任务和打印通道能力，供其他业务模块接入。
9. 标签实际展示字段完全读取打印模板配置，不在入库打印页面硬编码批次、SN、外部 SN 等字段。

## 4. 用户角色与场景

| 角色 | 典型场景 | 核心诉求 |
| --- | --- | --- |
| 仓库操作员 | 入库完成后打印并粘贴物料标签 | 操作简单、数量准确、快速连续打印 |
| 仓库管理员 | 批量打印、重复打印、查看记录 | 掌握累计打印数量，能够追溯异常 |
| 企业管理员 | 配置模板、打印机和权限 | 统一企业打印规则，控制敏感操作 |
| 实施/客服 | 协助客户配置打印环境 | 快速定位模板、服务和设备问题 |
| 审计/管理人员 | 查询打印记录 | 明确谁在何时因何原因打印了哪些标签 |

### 4.1 典型使用场景

1. 入库单详情打印：操作员完成一张入库单后，按参考待打印数量快速打印标签。
2. 分批打印：本次入库 100 件，先打印 40 张，剩余 60 张后续继续打印。
3. 列表批量打印：操作员勾选多张已入库单，统一预览并打印。
4. 再次打印：标签破损、丢失或打印不清晰时，可直接再次输入数量打印。
5. 多模板打印：普通物料使用标准标签，批次物料使用批次追溯标签。
6. 多打印机打印：不同仓库连接各自的标签打印机和本地打印服务。

## 5. 关键概念与口径

| 概念 | 定义 |
| --- | --- |
| 本次入库数量 | 当前入库单明细实际确认入库的物料数量。 |
| 已打印数量 | 历次打印任务的成功标签数量累计值，可大于本次入库数量。 |
| 参考待打印数量 | `max(本次入库数量－已打印数量, 0)`，仅用于默认带出和进度参考，不限制本次打印数量。 |
| 本次打印数量 | 用户本次希望生成的标签张数，必须为大于 0 的整数，可超过参考待打印数量。 |
| 打印任务 | 一次确认打印产生的任务，包含模板、打印机、业务单据和多条打印明细。 |
| 打印记录 | 打印任务及其执行结果的可查询历史。 |
| 一物一标 | 每 1 件物料生成 1 张标签。 |
| 标签模板 | 定义标签尺寸、显示字段、字段标题、位置、格式、条码规则和样式的配置，是标签内容的唯一展示依据。 |
| 打印通道 | 浏览器打印、本地打印服务或后续云打印服务等执行方式。 |

### 5.1 数量计算

```text
参考待打印数量 = max(本次入库数量 - 累计已打印数量, 0)
```

本次打印数量：

```text
本次打印数量为大于 0 的整数
本次打印数量可大于参考待打印数量
打印成功后：累计已打印数量 += 本次成功数量
```

数据存储口径：

- 业务明细只保存累计已打印数量 `printed_quantity`。
- 不单独保存待打印数量；参考待打印数量由查询时实时计算。
- 不保存“最大可打印数量”，打印提交不进行入库数量上限校验。

## 6. 产品范围

### 6.1 V1.0 范围

1. 入库单详情页打印入口。
2. 入库单列表批量打印入口。
3. 可打印物料选择、搜索、全选和清空。
4. 本次打印数量自由调整与正整数校验。
5. 标签模板选择，并按模板读取标签尺寸和条码/二维码配置。
6. 打印前标签预览和缩放。
7. 打印机与本地打印服务检测。
8. 创建打印任务、接收打印结果和更新已打印数量。
9. 打印记录查询。

### 6.2 后续范围

1. 可视化标签模板设计器。
2. 标签字段拖拽、字体、边框、图片和企业 Logo 配置。
3. 云打印服务。
4. 唯一码防重复扫码。
5. 仓位标签、箱码、托盘码及父子包装关系。
6. 打印审批和异常工单。
7. 移动端或 PDA 打印。

### 6.3 标签内容配置原则

1. 实际打印标签展示什么信息，完全由所选打印模板决定。
2. 打印页面不固定要求展示批次、SN、外部 SN、供应商等标签字段。
3. 业务模块提供当前打印对象可使用的完整字段数据，打印中心按模板配置读取所需变量。
4. 模板未配置的字段不渲染，即使业务数据中存在该字段。
5. 模板配置了字段但当前业务数据为空时，按模板的空值策略处理。
6. 条码或二维码的编码内容也从模板配置读取，可选择物料编码、批次、SN、外部 SN 或组合表达式。

说明：打印操作页为方便选择和核对，可固定展示物料名称、编码、本次入库数量、累计已打印数量等管理字段；这些字段不等于最终标签必须打印的内容。

## 7. 用户流程

### 7.1 入库单详情页打印

1. 用户进入状态允许打印的入库单详情页。
2. 点击“打印标签”。
3. 系统加载当前入库单和物料明细。
4. 系统默认勾选参考待打印数量大于 0 的物料。
5. 系统默认带出每种物料的参考待打印数量；参考值为 0 时，用户仍可手动选择并输入打印数量。
6. 用户选择物料并调整本次打印数量。
7. 用户进入预览页，选择模板和打印机；系统按模板加载标签尺寸、条码/二维码规则及字段布局。
8. 系统渲染标签并展示标签总数。
9. 用户确认打印。
10. 系统创建打印任务并发送至打印通道。
11. 本地打印服务返回结果，系统记录成功数和失败数。
12. 系统更新物料已打印数量并生成打印记录。

### 7.2 入库单列表批量打印

1. 用户在入库单列表勾选多张单据。
2. 点击“批量打印标签”。
3. 系统校验单据状态、仓库权限和可打印明细。
4. 系统按“入库单＋物料＋批次”展示可打印明细及累计已打印数量。
5. 不同入库单、不同批次默认不合并。
6. 用户调整数量后进入统一预览。
7. 系统创建一个批量打印任务，并保留各业务单据的明细关联。

### 7.3 再次打印

1. 用户重新进入入库单打印页面。
2. 系统展示累计已打印数量和参考待打印数量。
3. 用户选择需要再次打印的物料，并输入本次打印数量。
4. 系统不限制本次打印数量是否超过参考待打印数量。
5. 打印成功后，成功数量继续累计到已打印数量。
6. 每次打印均生成独立打印记录，便于追溯累计数量来源。

## 8. 功能需求

### F1 打印入口与状态控制

| 编号 | 需求 |
| --- | --- |
| F1-01 | 入库单详情页提供“打印标签”入口。 |
| F1-02 | 入库单列表支持勾选多张单据后“批量打印标签”。 |
| F1-03 | 默认仅“已入库”或业务确认完成的单据允许打印。 |
| F1-04 | 草稿、已作废、已取消单据不允许打印。 |
| F1-05 | 已红冲或已退回的业务数据是否允许再次打印，由业务模块返回可打印状态。 |
| F1-06 | 无打印权限时隐藏入口；直接访问打印地址时仍需后端鉴权。 |

验收标准：

- Given 入库单状态为已入库且用户有权限，When 查看详情，Then 展示“打印标签”入口。
- Given 入库单状态为草稿或已作废，When 用户尝试打印，Then 阻止并提示当前状态不可打印。
- Given 用户无打印权限，When 查看详情或列表，Then 不展示打印入口。

### F2 数据加载与物料选择

| 编号 | 需求 |
| --- | --- |
| F2-01 | 加载入库单号、仓库、入库日期、批次及物料明细。 |
| F2-02 | 物料明细展示物料名称、编码、规格、批次、本次入库、累计已打印和参考待打印数量。 |
| F2-03 | 支持按入库单号、物料编码和物料名称搜索。 |
| F2-04 | 支持全选、清空和逐项勾选。 |
| F2-05 | 参考待打印数量为 0 的物料默认不勾选，但允许用户手动勾选并输入打印数量。 |
| F2-06 | 列表批量打印按单据分组展示，并保留物料批次边界。 |

验收标准：

- Given 本次入库 5 件且累计已打印 2 张，When 打开打印页，Then 参考待打印数量展示为 3。
- Given 本次入库 5 件且累计已打印 8 张，When 打开打印页，Then 累计已打印展示为 8，参考待打印展示为 0，仍允许继续打印。
- Given 多张入库单存在相同物料但批次不同，When 批量打印，Then 分别展示，不自动合并。
- Given 搜索物料编码，When 输入完整或部分编码，Then 返回匹配物料。

### F3 打印数量

| 编号 | 需求 |
| --- | --- |
| F3-01 | 本次打印数量默认等于参考待打印数量。 |
| F3-02 | 用户可将打印数量调整为大于 0 的整数。 |
| F3-03 | 本次打印数量允许超过参考待打印数量，不设置业务数量上限。 |
| F3-04 | 打印数量改为 0 时，自动取消该物料选择。 |
| F3-05 | 提交打印前校验打印数量为正整数，并记录提交时的本次入库数量和累计已打印数量快照。 |
| F3-06 | 多人并发打印时，各任务独立执行；成功数量均累计到已打印数量。 |

验收标准：

- Given 参考待打印数量为 3，When 用户输入 4，Then 允许提交并创建 4 张标签的打印任务。
- Given 参考待打印数量为 0，When 用户手动选择物料并输入 2，Then 允许提交；成功后累计已打印数量增加 2。
- Given 两名用户同时打印同一物料，When 两个任务均成功，Then 分别累计两个任务的成功数量。

### F4 模板及打印内容配置

| 编号 | 需求 |
| --- | --- |
| F4-01 | 支持选择启用状态的标签模板。 |
| F4-02 | 默认带出仓库、物料类别或企业配置的默认模板。 |
| F4-03 | 模板至少包含模板名称、标签尺寸、字段定义、字段显示开关、编码方式和适用业务。 |
| F4-04 | V1.0 预置标准物料标签、仓库简洁标签、批次追溯标签。 |
| F4-05 | 支持条码、二维码；编码类型、编码值来源和显示位置均读取模板配置。 |
| F4-06 | 模板支持配置常用尺寸，如 50×30 mm、60×40 mm；打印页面不单独覆盖模板尺寸。 |
| F4-07 | 条码值为空、超长或不符合编码规范时，预览前提示异常。 |
| F4-08 | 标签字段不在打印页面写死，预览和打印时直接读取模板字段列表及布局配置。 |
| F4-09 | 模板可配置是否显示批次号、SN、外部 SN、供应商、生产日期、有效期等字段。 |
| F4-10 | 模板可配置字段标题、变量来源、格式、排序、坐标、宽高、字体、对齐方式和空值策略。 |
| F4-11 | 用户切换模板后，系统按新模板重新读取字段并渲染预览。 |
| F4-12 | 打印页面只消费模板配置，不在打印页面重复维护字段显示开关。 |

模板可选字段：

| 字段 | 模板变量示例 | 说明 |
| --- | --- | --- |
| 物料名称 | `material.name` | 模板配置后才显示 |
| 物料编码 | `material.code` | 可作为文本、条码或二维码数据源 |
| 规格型号 | `material.specification` | 来源于物料或单据快照 |
| 单位 | `material.unit` | 基本单位或入库单位 |
| 批次号 | `batch.no` | 模板决定是否显示 |
| SN 码 | `serial.sn` | 内部序列号，模板决定是否显示及编码 |
| 外部 SN 码 | `serial.externalSn` | 供应商或外部系统序列号 |
| 入库单号 | `business.no` | 入库业务字段 |
| 仓库 | `warehouse.name` | 当前入库仓库 |
| 入库日期 | `business.confirmedAt` | 业务确认日期 |
| 供应商 | `supplier.name` | 采购入库适用 |
| 生产日期 | `batch.productionDate` | 批次数据可用时提供 |
| 有效期 | `batch.expiryDate` | 批次数据可用时提供 |
| 企业 Logo | `company.logo` | 图片变量 |

#### F4.1 字段配置结构

每个模板字段至少支持：

| 配置项 | 说明 |
| --- | --- |
| `visible` | 是否显示 |
| `label` | 字段标题，如“外部 SN” |
| `variable` | 数据变量，如 `serial.externalSn` |
| `type` | 文本、条码、二维码、图片、日期 |
| `format` | 日期、数字或文本格式 |
| `x` / `y` | 在标签中的位置 |
| `width` / `height` | 字段区域尺寸 |
| `fontSize` / `fontWeight` | 字体样式 |
| `align` | 左、中、右对齐 |
| `emptyPolicy` | 空值时隐藏、留空、显示默认值或阻止打印 |
| `defaultValue` | 空值默认内容 |

模板渲染规则：

1. 打印中心读取模板当前版本的字段配置。
2. 仅处理 `visible=true` 的字段。
3. 根据 `variable` 从打印上下文中读取数据。
4. 按字段类型和格式生成文本、条码、二维码或图片。
5. 按空值策略处理缺失数据。
6. 使用同一模板版本完成预览和实际打印。

#### F4.2 SN 与外部 SN 规则

1. SN 和外部 SN 都是模板可选变量，不是所有标签的固定字段。
2. 当模板未配置 SN 或外部 SN 时，标签不展示对应信息。
3. 当模板配置 SN 类变量时，业务模块需按单件提供序列号数据。
4. 一次打印多张标签时，每张标签绑定对应的 SN/外部 SN，不得把多个 SN 混在同一张“一物一标”标签中。
5. 若打印数量大于可用 SN 数量：
   - 模板将 SN 配置为必填时，阻止预览或打印，并提示缺少的序列号数量。
   - 模板将 SN 配置为非必填时，超出部分按空值策略渲染。
6. 内部 SN 与外部 SN 可同时显示，也可分别作为条码或二维码的数据源。

## 9. 异常与边界

| 场景 | 处理要求 |
| --- | --- |
| 入库单状态已变化 | 提交前阻止打印并提示刷新 |
| 多人同时打印同一物料 | 各任务独立执行，成功数量分别累计 |
| 模板必填变量为空 | 阻止预览或打印，并提示具体缺失字段 |
| 模板非必填变量为空 | 按字段 `emptyPolicy` 隐藏、留空或显示默认值 |
| 打印数量大于 SN 数据数量 | SN 必填时阻止打印；SN 非必填时超出部分按空值策略处理 |
| 条码内容超出模板宽度 | 预览提示并阻止打印或自动切换规则 |
| 本地打印服务未启动 | 显示安装/启动指引，不创建成功记录 |
| 打印机离线或缺纸 | 根据设备回执标记失败，支持重试 |
| 网络中断 | 保留待发送任务，恢复后允许继续发送 |
| 部分标签失败 | 成功部分计数，失败部分可单独重试 |
| 用户重复点击 | 通过按钮禁用和幂等键避免重复任务 |
| 模板打印期间被修改 | 当前任务使用创建时锁定的模板版本 |
| 单据后续退货或红冲 | 不直接删除历史打印记录，业务页面标记来源状态 |
| 打印数量非常大 | 超过阈值时拆分批次或转后台任务 |

## 10. 非功能需求

### 10.1 性能

1. 100 条物料明细的打印数据加载时间建议不超过 2 秒。
2. 500 张标签预览应采用分页或分段渲染，首屏建议不超过 3 秒。
3. 超过 1000 张标签的任务建议转后台生成，并展示进度。

### 10.2 安全

1. 打印任务创建和模板管理必须进行后端权限校验。
2. 本地打印服务应校验企业身份和任务签名，防止其他网页调用。
3. 模板变量按白名单输出，变量值转义后渲染，防止脚本注入。
4. 打印记录应保留操作人、时间、设备和业务来源。

### 10.3 可用性

1. 打印前明确展示标签总数，避免误操作。
2. 异常提示需说明原因和处理方式。
3. 操作员返回修改后，应保留已选物料和数量。
4. 键盘、扫码枪输入不得触发重复提交。

## 11. 埋点与成功指标

### 11.1 埋点

- 打印入口点击次数
- 详情页/列表批量入口占比
- 预览成功和失败次数
- 创建打印任务次数
- 打印成功率、部分成功率和失败率
- 平均每次打印标签数
- 同一物料重复打印次数
- 打印机离线、服务未启动、模板异常次数
- 从进入打印页到任务创建的平均耗时

### 11.2 成功指标

1. 试点仓库 90% 以上入库标签通过系统直接生成。
2. 每次成功打印数量均准确累计，打印记录与累计已打印数量一致。
3. 打印任务可追溯率达到 100%。
4. 本地打印服务在线时，打印任务成功率达到 98% 以上。
5. 操作员完成一次普通入库打印的平均操作时间不超过 2 分钟。
6. 后续新业务接入打印中心时，通用能力复用率达到 70% 以上。

## 13. 验收标准汇总

| 编号 | 场景 | 验收结果 |
| --- | --- | --- |
| AC-001 | 详情页打印 | 已入库单据可进入打印页并加载正确明细 |
| AC-002 | 数量展示 | 本次入库 5、累计打印 2 时，参考待打印为 3 |
| AC-003 | 超参考数量打印 | 参考待打印为 3 时输入 4，允许创建 4 张标签 |
| AC-004 | 分批打印 | 首次打印部分数量后，累计已打印和参考待打印数量正确更新 |
| AC-005 | 批量打印 | 多张单据可统一打印且保留单据、批次边界 |
| AC-006 | 重复打印 | 参考待打印为 0 时仍可继续输入数量打印 |
| AC-007 | 并发累计 | 多人并发任务成功后，成功数量均正确累计 |
| AC-008 | 权限 | 无权限用户不能通过页面或接口打印 |
| AC-009 | 模板字段显示 | 模板关闭批次、SN 或外部 SN 后，预览和打印均不出现对应字段 |
| AC-010 | 模板字段取值 | 模板开启外部 SN 后，每张标签读取其对应的 `serial.externalSn` |
| AC-011 | 模板切换 | 切换模板后按新模板字段配置重新渲染，不沿用旧模板显示字段 |
| AC-012 | 必填字段缺失 | 模板必填变量无数据时阻止打印，并提示缺失字段 |
